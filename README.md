# DiamondGuru
The DiamondGuru Programming Language. (Under Development)

### How to run -
```shell
$ git clone https://github.com/DuskyElf/DiamondGuru.git
```
```shell
$ cd src
```
```shell
$ python3 DGuru.py
```

### Goals (features)
- Object Oriented
- Dynamicly typed
- Compiled
- Strict type checking on compile time
